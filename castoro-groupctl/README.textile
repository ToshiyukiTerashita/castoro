h1. castoro-groupctl

castoro-groupctl is a remote control subsystem for peer groups of Castoro.

h2. Installation

<pre>
gem install --no-rdoc --no-ri castoro-groupctl-0.0.1.pre1.gem
cd /usr/local/lib/ruby/gems/1.9.1/gems/castoro-groupctl-0.0.1.pre1/etc/init.d
make install
cd /usr/local/lib/ruby/gems/1.9.1/gems/castoro-groupctl-0.0.1.pre1/etc/castoro
cp groupctl.conf-sample-ja.conf /etc/castoro/groupctl.conf
vi /etc/castoro/groupctl.conf
</pre>
